2 Input/Output tools

Defined in base.lisp.

  • [variable] *ITER-ABC-TREE-RAW* TREE-ITERATOR

    A generator that yields ABC trees from *STANDARD-INPUT*.

  • [variable] *PARSE-ABC-CAT-ANNOTED* CAT-ITERATOR

    The parser of ABC categories with meta features.

  • [variable] *ALTER-PARSE-ABC-TREE-NODES* TREE-MODIFIER-DESTRUCTIVE

    Parse each non-terminal node of a given tree as an AMOOVE/CAT:CAT wrapped by AMOOVE/ANNOT:ANNOT.

    Note that this function is destructive and modifying in-situ.

  • [function] PPRINT-ABC-NODE ITEM

    A helper function that pretty print an AMOOVE/CAT:CAT wrapped by AMOOVE/ANNOT:ANNOT.

3 Movement of comparative nodes

  • [function] MAKE-YORI-LEXSPEC O

  • [symbol-macro] LEXSPEC-YORI

  • [function] CONVERT-PREJ TREE-PREJ CAT-ROOT-ORIGINAL O

    Type-shift prejacent constituents of comparatives.

    E.g.

    (<S/S> (NP 太郎)
       (NP\<S/S>> より))

    ↓

    (S|NP|<S/S>|<S/S> (NP 太郎) 
                  (S|NP|<S/S>|<S/S>|NP より))
